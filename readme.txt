退職後の趣味として囲碁や将棋のように自分はプログラミングを選びました。
各言語の知識は入門書レベルです。アドバイスが欲しくて公開しました。

有名な書籍「実践テスト駆動開発」のセクション事の進捗をSnapShotsフォルダに納めました。
本書では、自明な事として省略されたインポート宣言などもライブラリ事に細かく宣言しました。
ただし、OpenFireは別途ダウンロードして本書に沿って設定する必要があります。

後半では、独自に設計判断したドメイン処理を自分のしっている言語で実装しました。
各言語に関してはダウウンロードとパス設定が必要です。

また、相対パスが可能な部分は行っているけど、ダメな部分はA:/PJ/sniperにプロジェクトを置く必要があります。

OCaml C++ C に関しては、自分の知見ではヘッダーファイルとリンクによる方法ではなく、ソースレベルでの展開になっています。
